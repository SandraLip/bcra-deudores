⚙️ Configuración inicial (solo una vez)

  1. Entrá a script.google.com con tu cuenta de Google
  2. Hacé clic en Nuevo proyecto
  3. Borrá todo el contenido y pegá este código:
    function doGet(e) { var cuit = e.parameter.cuit; var url = "https://api.bcra.gob.ar/centraldedeudores/v1.0/Deudas/" + cuit; var resp = UrlFetchApp.fetch(url, {muteHttpExceptions: true}); return ContentService .createTextOutput(resp.getContentText()) .setMimeType(ContentService.MimeType.JSON); }
  4. Hacé clic en Implementar → Nueva implementación
